# S

硫-S，位于周期表第三周期，第IVA族的非金属，常见价态有：-2、0、+4、+6。在自然界中以硫黄等形式存在游离态。

用途：生产硫酸、硫肥、农药、火柴、含硫橡胶、肥皂等

---

### S单质

俗称硫黄

是一种黄色晶体，质脆，易研成粉末。

难溶于水，微溶于酒精，易溶于$\ce{CS2、CCl4}$等非极性溶剂

存在多种同素异形体（$\ce{S2、S4、S6、S8}$等），$\ce{S8}$最稳定

具有氧化性与还原性（氧化性<$\ce{Cl2}$）

* 以$\ce{FeS2}$制取硫单质:

$\ce{3FeS2 + 12C + 8O2 \xlongequal{点燃} Fe3O4 + 12 CO + 6S}$

* 硫可与除$\ce{Au、Pt}$外的几乎所有金属在加热条件下直接化合;
  
  $\ce{Hg + S \xlongequal{(\Delta)} HgS}$

——$\ce{HgS}$：朱砂

- 只能将变价金属氧化到低价。
  
  $\ce{Fe + S \xlongequal{\Delta} FeS}$
  
  $\ce{2Cu + S \xlongequal{\Delta} Cu2S}$
* 硫一般可以与除稀有气体，碘、氮以外的非金属单质直接化合
  
  $\ce{O2 + S \xlongequal{点燃} SO2}$
  
  $\ce{C + 2S \xlongequal{\Delta} CS2}$
  
  $\ce{H2 + S \xlongequal{\Delta} H2S}$

* 硫可被氧化性酸氧化
  
  $\ce{S + 2HNO3(浓) = H2SO4 + 2NO\uparrow }$
  
  $\ce{S + 2H2SO4(浓) \xlongequal{\Delta} 3SO2\uparrow + 2NO\uparrow}$

* 硫在碱性条件下歧化到-2和+4价，加热可加快反应
  
  $\ce{3S + 6NaOH \xlongequal{\Delta} 2Na2S + Na2SO3 + 3H2O}$
  
  ^ $\ce{3S + 6OH- \xlongequal{\Delta} 2S^2- + SO3^2- + 3H2O}$

* 烟花
  
  $\ce{S + 2KNO3 + 3C \xlongequal{点燃} K2S + N2\uparrow + 3O2\uparrow}$

### $\ce{SO2}$

一种无色，有刺激性气味的有毒气体，酸性氧化物，易溶于水(1：40)

硫处于中间价态(+4)，同时具有氧化性和还原性，硫酸型酸雨的源头

- 食品添加剂
  
  少量添加到食品中起漂白，杀菌，抗氧化等作用

- 漂白性
  
  与有色物质化合生成不稳定的无色物质，加热后可复原，且不能漂白指示剂，可用于漂白纸张、毛、丝等。

- 溶于水
  
  $\ce{SO2 + H2O <=> H2SO3}$

- 制备$\ce{SO3}$

    $\ce{2SO2 + O2 <=>[V2O5][\Delta] 2SO3}$

- 实验室制法
  
  $\ce{Na2SO3_{（粉末）} + H2SO4(浓) = Na2SO4 + SO2\uparrow + H2O}$
  
  浓硫酸要用70%质量分数的（加快反应+防止溶液中溶解太多$\ce{SO2}$）

- 亿些氧化还原
  
  $\ce{SO2 + 2H2S = 3S(\downarrow) + 2H2O}$ （沉淀符号看情况）
  
  $\ce{5SO2 + 2MnO4- + 2H2O = 2Mn^2+ + 5SO4^2- + 4H+}$ 
  与酸性高锰酸钾反应，用于吸收$\ce{SO2}$
  
  $\ce{SO2 + Cl2 + 2H2O = H2SO4 + 2HCl}$
  
  $\ce{SO2 + Br2 + 2H2O = H2SO4 + 2HBr}$
  
  $\ce{SO2 + I2 + 2H2O = H2SO4 + 2HI}$
  
  $\ce{2Fe^3+ + SO2 + 2H2O = 2Fe^2+ + SO4^2- + 4H+}$
  
  $\ce{H2O2 + SO2 = H2SO4}$
  
  $\ce{Na2O2 + SO2 = Na2SO4}$
  
  $\ce{3SO2 + 2H2O + 3Ba^2+ + 2NO3- = 3BaSO4\downarrow + 2NO\uparrow + 4H+}$
  
  ……

- “脱硫”
  
  $\ce{CaO + SO2 = CaSO3}$
  
  $\ce{2CaSO3 + O2 = 2CaSO4}$

- 与量相关
  
  $\ce{SO2 + OH- = HSO3-}$
  
  $\ce{SO2 + 2OH- = SO3^2- + H2O}$
  
  $\ce{SO2 + Ca(OH)2 = CaSO3\downarrow + H2O}$
  
  $\ce{2SO2 + Ca(OH)2 = Ca(HSO3)2}$
  
  $\ce{SO2 + 2NH3·H2O = (NH4)SO3 + 2H2O}$
  
  $\ce{SO2 + NH3·H2O = NH4(HSO3)}$

### $\ce{H2SO4}$

  重要的化工原料，工业用途广泛。酸酐为$\ce{SO3}$，难挥发性强酸，浓硫酸具有强氧化性(稀硫酸没有)，无色，油状液体，沸点高。

##### 浓硫酸

  较活泼的金属（浓/稀都反应）的可能会发生2个反应

- 吸水性
  
  将游离的$\ce{H2O}$分子和结晶水合物中的$\ce{·H2O}$吸收的能力
  
  会导致敞口旧置的浓硫酸质量增加，浓度下降
  
  可作酸性干燥剂（不可干燥$\ce{SO2、NH3、HI、HBr}$等，但可干燥$\ce{SO2}$）

- 脱水性
  
  将有机物中的氢氧原子以$\ce{H2O}$的比例(1:2)脱出的能力
  
  会导致有机物炭化
  
  $\ce{CuSO4·5H2O ->[H2SO4(浓)] CuSO4}$ //化学变化

- 强氧化性
  
  会将变价金属氧化到高价，一般需要加热
  
  常温下与铁/铝反应生成致密的氧化物薄膜（与液氯相似），阻止了进一步反应（钝化），因此可在常温下使用铁/铝制容器储存、运输浓硫酸。
  
  **加热条件下照常反应**
  
  $\ce{C + 2H2SO4(浓) \xlongequal{\Delta} CO2\uparrow + 2SO2\uparrow + 2H2O}$
  
  $\ce{2H2SO4(浓) + Cu \xlongequal{\Delta} CuSO4 + SO2\uparrow + H2O}$
  
  $\ce{H2S + H2SO4(浓) = S\downarrow + SO2\uparrow + 2H2O}$
  
  $\ce{S + 2H2SO4(浓) \xlongequal{\Delta}3SO2\uparrow + 2H2O}$

- 难挥发性（制挥发性酸）
  
  使用稀硫酸制备
  
  可用其制盐酸、硝酸
  
  但不能制氢溴酸、氢碘酸等会与其反应的酸（使用$\ce{H3PO4}$ Instead）
  
  $\ce{2NaCl_{(S)} + H2SO4 \xlongequal{\Delta}2HCl\uparrow + Na2SO4}$

  ### $\ce{SO4^2-}$

- 硫酸盐
  
  $\ce{CaSO4·2H2O}$ //石膏，$\ce{CaSO4}$在自然界的主要存在形式
  
  $\ce{2CaSO4·H2O}$ //熟石膏，石膏加热失去大部分结晶水而来
  
  $\ce{BaSO4}$ 
  
  重晶石，生产其他钡盐的主要原料，不溶于水和酸，且不易于被X射线透过，可用于消化道检查，即“钡餐”
  
  $\ce{CuSO4}$ 
  
  白色粉末，极易吸水变成蓝色晶体$\ce{CuSO4·5H2O}$（胆矾），因此常用于检测水(**但不用于干燥**)；可与石灰乳混合制成农药波尔多液
  
  $\ce{SrSO4}$ //天青石
  
  $\ce {ZnSO4·7H2O}$ //皓钒
  
  $\ce{CuSO4·5H2O}$ //胆矾
  
  $\ce{K[Al(SO4)]·12H2O}$ //明矾
  
  $\ce{FeSO4·7H2O}$ //绿矾

- 硫酸根离子的检验
  
  取适量待测液于试管中，加入足量稀盐酸，若无明显现象（有沉淀再取上层清液）加入$\ce{BaCl2}$溶液，若出现白色沉淀，则证明原溶液中存在$\ce{SO4^2-}$

  ### $\ce{H2S}$
  
  无色，有臭鸡蛋气味的有毒气体，溶解度1:2.6
  
  二元弱酸+强还原剂

- 电离
  
  $\ce{H2S <=> H + +HS-}$
  
  $\ce{HS- <=> H+ + S^2-}$

- 离子反应
  
  $\ce{H2S + CuSO4 = H2SO4 + CuS\downarrow}$
  
  $\ce{CuS}$是不溶于酸和水的黑色固体

- 氧化还原
  
  $\ce{5H2S + 2MnO4- = 5S\downarrow + 2H+ + 2Mn^2+ + 4H2O}$
  
  ……
  
  - 连续氧化
    
    $\ce{H2S ->[O2]S->[O2]SO2->[O2]SO3->[H2O]H2SO4}$
